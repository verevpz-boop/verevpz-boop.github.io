# Спутник «Сигнатурная голова» — говорящий real-time аватар

> Статус: **ФАЗА ДОКУМЕНТОВ** (порядок крупных фич: документы → макет → утвердить → код).
> Утверждено Pavel'ом как большой проект 2026-06-04. Код НЕ трогаем до согласования макета.

## 1. Что это
Новая страница-«спутник» сайта: **сигнатурная (wireframe/low-poly) голова на чёрном звёздном фоне, которая в реальном времени слушает голос, отвечает голосом и шевелит губами (липсинк). Можно перебить голосом.**

## 2. 🔴 Хард-правила (нерушимые, от Pavel'а)
1. **Башка рендерится с липсинком в реалтайме** — рот двигается синхронно с речью.
2. **Перебивание голосом** — заговорил поверх → голова мгновенно затыкается и слушает.
3. **Латентность ≤ 1 секунды** — от «ты замолчал» до «голова начала отвечать». Это ПОТОЛОК, не цель.
4. **Всё бесплатно** — хостинг, STT, LLM, TTS, рендер. 0₽ на инфру.
5. Может быть «тупой» по содержанию — но скорость и живость обязательны.

## 3. Архитектура пайплайна

```
  ┌─────────┐   голос    ┌──────────────┐  текст   ┌─────────────┐ токены ┌──────────────┐
  │  Микро  │ ─────────► │ STT (браузер)│ ───────► │ LLM-прокси  │ ─────► │ TTS (браузер)│
  │  фон    │            │ Web Speech   │          │ → Cerebras  │ стрим  │ speechSynth  │
  └─────────┘            └──────┬───────┘          └─────────────┘        └──────┬───────┘
       ▲                        │ interim (VAD)                                  │ аудио + амплитуда
       │ перебил                ▼                                                ▼
       │                 ┌─────────────┐                                  ┌──────────────┐
       └──────────────── │ Барж-ин:    │ ◄─────────────────────────────── │ R3F голова   │
         cancel TTS +    │ стоп всего  │        громкость → рот            │ wireframe    │
         abort LLM       └─────────────┘                                  └──────────────┘
```

**Поток:** мик → браузерный STT (распознаёт на лету) → по паузе финальный текст → прокси → Cerebras (стрим токенов) → браузерный TTS озвучивает первое предложение, пока LLM дописывает → голова открывает рот по громкости аудио. Если пользователь заговорил во время ответа (VAD по interim-результатам STT) → `speechSynthesis.cancel()` + abort fetch → слушаем заново.

| Компонент | Решение | Стоимость | Заметка |

## 5. Бюджет латентности (доказательство ≤1с)

| Шаг | Время |
|---|---|
| STT финализирует фразу (по паузе) | ~150 мс |
| Сеть до Worker + Cerebras TTFT | ~200–350 мс |
| Первое аудио TTS (браузер) | ~100 мс |
| **ИТОГО до первого слова головы** | **~450–600 мс** ✅ < 1с |
| Перебивание (стоп речи) | < 50 мс ✅ |

Ключ к скорости: **стриминг** — голова произносит первое предложение, пока LLM генерит остальное. Cerebras на 2600 ток/с снимает LLM как узкое место.

## 6. Перебивание (барж-ин) — дизайн
- Web Speech STT работает НЕПРЕРЫВНО и отдаёт промежуточные результаты.
- Как только пришёл interim-результат во время ответа головы → триггер «перебили»:
  1. `window.speechSynthesis.cancel()` — голова замолкает (<50мс).
  2. `AbortController.abort()` — обрываем стрим LLM.
  3. Сбрасываем состояние головы в «слушаю».
- Никакого серверного VAD — всё в браузере.

## 7. Голова + липсинк — дизайн
- **Геометрия:** низкополигональная/wireframe голова (GLB или процедурная), «сигнатурный» стиль — лёгкая для онлайн-рендера, в духе глобуса сайта.
- **Фон:** чёрный звёздный (переиспользуем подход со звёздами/космосом главной).
- **Липсинк v1 (просто):** AnalyserNode на выходе TTS → RMS-громкость → открытие челюсти (morph target или поворот кости). Дёшево, читается как речь.
- **Липсинк v2 (точнее):** виземы по фонемам/таймингам — позже, если v1 мало.
- **Референс лица:** Pavel упоминал, что скидывал лицо с липсингом из минимальных сигнатур — найти и приложить к макету.

## 9. Единственный подводный камень
Статический сайт не может дёргать LLM-хаб без засветки ключа.
- **Локально/демо:** работает сразу (как чат мишки через localhost).
- **Публично:** нужен Cloudflare Worker (free) с ключом на стороне Worker'а. Это и есть «прокси, которого пока нет».
