## 🔴 ДЖАРВИ — ЧТО ОСТАЛОСЬ
- **🎙 УШНОЙ ТЕСТ ГОЛОСА (за Pavel'ом)** — голос вшит (см. СДЕЛАНО п.7), но реальный разговор+перебивание машинно не проверить. Pavel: хард-рефреш → клик по хосту → разрешить мик → говорить. Если глухо/эхо — чинить по `JARVI_BUILD_ROADMAP.md`.
- **🔴 «Микрофон пропадает» = выжигание WebGL-контекста** (грабля хэндоффа): хост-канвас — один из многих R3F-канвасов страницы; при долгой сессии/множестве рефрешей/рестартов дев-сервера он теряет контекст и рисует пусто (значок и аватар «исчезают»). На ЧИСТОЙ загрузке значки стабильно на месте (проверено скринами). Если будет пропадать у реальных юзеров (не в дев-churn) — харднуть (меньше канвасов / context-менеджмент). Не из-за кода значков.
- **Значки = плоские иконки billboard-СПРАЙТАМИ** (НЕ 3D-примитивы — те читались как «кубок/болванка», Pavel забраковал). Рисуются на canvas (`makeMicTexture`/`makeRingTexture`/`makeSparkTexture`) в БЕЛОМ, тинт через `spriteMaterial.color`; микрофон = капсула-грилья+полоски+ярмо+ножка+подставка (как на демке-виджете). Размер от высоты ГОЛОВЫ: `anchor.s = headH`, спрайты — доли от него (микрофон ~0.64*headH общий ≈ капсула ~1/3 головы; комета-кольцо 0.5*headH). `anchor.y = top+0.07*H`, `x` = кость головы. 🔧 Канвас-слой 56vh (`pointer-events-none`, клик-зона 38vh), `fillFrac 0.53` — значок не срезается. Размер крутится одним числом-долей в JSX.
- **Безмолвное «привет-махание» при появлении** (Pavel склонялся к ДА, не финализировал): на маунте проиграть greeting + look-at, без звука; голос — с первого клика.
- **Размер/позиция хоста по глазу:** «чтоб читался» (десктоп ок; на телефоне его нет — Pavel принял). На главной стоит близковато к двери СТУДИЯ — подвинуть/уменьшить box.
- Мобила: хоста нет (десктоп-онли). Статичный фоллбэк — опц.
