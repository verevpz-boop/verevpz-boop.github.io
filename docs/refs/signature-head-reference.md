# Сигнатурная голова — РЕФЕРЕНС (от Pavel'а, 2026-06-05)

> Pavel прислал этот референс (скрин из ChatGPT, подпись «Set of three dimensional
> heads on abstract background»). Это канонический ориентир внешнего вида головы.
> Если нужен пиксельный файл — положить PNG рядом в `docs/refs/` под этим именем.

## Что на референсе
- **4 wireframe-головы** человека (лысые, без волос), расставлены сеткой 2×2.
  - верх-лево — **профиль** (лицом вправо)
  - верх-право — **фронт/три четверти**
  - низ-лево — три четверти
  - низ-право — фронт со спины/анфас
- **Геометрия:** низкополигональная **quad-mesh сетка** (четырёхугольники), видны рёбра —
  классический «каркас». Не точки, не сплошные грани — именно **wireframe-сетка**.
- **Цвет:** ярко-электрический **синий** (#1E6BFF / голубое свечение), эмиссивные линии,
  **светящиеся глаза**. Фон почти чёрный.
- **Фон (abstract / tech):**
  - вертикальный **цифровой дождь** (падающие синие штрихи, «матрица»)
  - **светящаяся горизонтальная дуга** через середину (свечение-горизонт)
  - мелкие круглые «радар/таргет»-точки, рассыпанные по полю
  - лёгкая сетка-пол снизу
- **Настроение:** холодное, технологичное, премиальное, голографическое.

## Выводы для нашей головы
- **Стиль = WIREFRAME quad-mesh** (вопрос «каркас/точки/грани» референс закрывает в пользу каркаса).
- Глаза **светятся**.
- Фон головы (в окне 16:9) = тёмный + цифровой дождь + дуга-горизонт + таргет-точки.
- **ЛИПСИНК обязателен и должен быть ЗАМЕТЕН** (Pavel: «нужно липсинк на ней обозначить» —
  рот/нижняя челюсть сетки явно двигается под речь).
